Mecanicas

Rotar camara

Desliza tu dedo en la vista del mapa horizontalmente para rotar la camara y visualizar todo a tu alrededor.

Zoom

Desliza tu dedo verticalmente para acercar o alejar la camara de tu personaje.

Recolecta objetos

Presiona sobre los objetos que aparecen en el mapa para hacerte con ellos.

Menu de personaje

Presiona sobre el icono de personaje para acceder al menu de personaje.

Equipar arma principal

Presiona sobre el icono mas alejado para abrir el menu de armas principales y selecciona el arma que quieres equipar.

Equipa objetos secundarios

Presiona sobre los iconos en fila para abrir el menu de objetos secundarios y selecciona el objeto que quieres equipar en ese espacio.

Visualizar atributos

Presiona sobre el boton de atributos para visualizar los atributos de tu personaje, en este menu tambien puedes mejorar tus atributos.

Iniciar combates

Presiona sobre los enemigos que aparecen en el mapa para iniciar eventos de combate.

Muevete en combate

Presiona sobre la zona sobreada y desliza tu dedo para moverte en esa direccion.

Atacar

Desliza tu dedo en una de cuatri direcciones para realizar un ataque, secuencia de ataques resultan en combos! aqui una lista de los combos:

Espada rapida: Izquierda + Derecha

Ataque pesado: Izquierda + Derecha + Arriba

Furia: Izquierda + Izquierda + Derecha + Abajo

Abrir menu de objetos

Presiona sobre el boton de menu de objetos secundarios, aqui encontraras todos los objetos que equipaste antes del combate, tambien puedes agitar el movil para abrir o cerrar este menu. Una vez el menu este abierto, puedes alternar entre ambas filas deslizando tudedo horizontalmente sobre el menu.

Usar objetos del menu de objetos secundarios

Debes presionar sobre el objeto que quieres usar hasta que el circulo este lleno, el juego te indicara con una flecha hacia arriba que el objeto esta listo para usarse, una vez listo, desliza tu dedo en la direccion que el juego te indica para usar el objeto.
